# Inventory Form Automation Project
This is the document that will explain to us how the project works

## Purpose of this project
* We are testing an Inventory form from https://ndosisimplifiedautomation.vercel.app/#overview . It is a website built for users to practice and learn automation

## Software required for this project
* Intellij
* JDK

## Maven Dependencies required
* Selenium - latest version
* Testng
    * latest version
* apache.poi
* extentreports

## Design Pattern
* Page Object Model

## High level Test Cases
* User visit site
* Logs on to the site
* Complete an Order on an inventory form
* Verifies order
* Completes other field like shipping method, discounts and warranty
* Confirm order is complete
* Checks invoice
* Close Browser

## Running the test
* Go to GitHub
* Clone the repository onto your machine
* Confirm that all dependencies are available
* Run testng.xml

## Reporting

* Open index file on report folder using your browser of choice

## Proof
* Navigate to screenshots folder to get screenshots as proof
